Reported Whale Strandings on U.S. Coasts (2005-2015) (link)▼

    Interactive Web Map

    Focuses: Efficiently and succinctly communicate a large amount of information in an approachable manner; maximize number and variety of interactive features; NOAA Branding Guide (fonts, colors, images)

    Data Acquisition: Downloaded spreadsheet of strandings from NOAA, NetCDFs from AquaMaps models of species distributions.
    ArcGIS Pro: Worked with stranding data (points, spatiality based on lat/lon of stranding) to make toggling in HTML more feasible. Converted NetCDFs to rasters, selected cells with >50% probability, and converted to polygons. Export as GeoJSONs.
    HTML, CSS, JavaScript: Worked with MapBox to initialize an iFrame that allows interaction with the map and supplementary features on top of the map distinct. Site has an informational panel with pictures of each species, the sounds they make, and links to their information pages on NOAA's website. Map is interactive (panning and zooming) and data layers can be toggled on and off by species. Colors and text set to match NOAA branding guide where possible.




Description, Comparison, and Prediction of Pacific Lamprey Entosphenus tridentatus and Western Brook Lamprey (Lampetra ayresii) Habitat in the Southern Oregon Coastal Range▼

    Undergraduate Thesis

    Focuses: Translating GIS skills from class exercises to research in my field

    Data Acquisition: eDNA presence/absence dataset provided to me by South Slough NERR; Collected open source raster data describing pertinent environmental descriptors within the study area.
    ArcGIS Pro: Used this data to create raster layers for the remaining variables of interest including slope, hillshade, aspect, and cost distance. Reclassified data to be best suited for analysis and modeling.
    Python: Pulled values associated with the coordinates of sample sites from each raster. Used these values to train a random forest classifier to predict the likelihood of lamprey presence based on these variables. Drew conclusions about the significance of each variable in habitat preference based on Mean Decrease in Impurity (MDI).
    Output: 10m raster displaying suitability predictions made by random forest classifier for each species, MDI measurements indicating relative significance of variables, general report of findings for the public, 50-page technical report, oral defense
